## Features

1. PWA with offline support and installability
2. Firebase backend already configured
3. Fully responsive and customizable
4. Image optimization
5. Full SEO friendly

## Development

```sh
git clone
cd nextjs-artist-portfolio
yarn install
yarn dev
```
